その2 簡単な計算を行う

四則計算をしたり割り算の余りを求めたりします。

必要な知識

  • 四則計算の方法
  • 余り（mod）を求める方法
  • プログラムを組むときの整数と小数の扱いの違い
    • 例えば、整数 / 整数は切り捨てで整数になることが多いです
      • 説明は面倒なのでググったりしてください

問題1

2つの整数（x、 y）を受け取って、和・差・積・商・剰余をそれぞれ求めましょう。

注意

  • 恐らく、yが0だと商を求めるところでエラーになります。

rubyでの実行例

$ ruby ex1。rb
x = 5
y = 3
x + y = 8
x - y = 2
x + y = 15
x / y = 1
x mod y = 2
$

問題2

以下の例のように、体重(kg)と身長(m)を入力して、BMI (Body Mass Index)を計算するプログラムを書きなさい。

ヒント

  • BMI = 体重 /（身長の2乗）
  • 整数ではなく小数を受け取るようにしましょう（型のある言語の場合）
  • 整数が入力された場合も小数で結果を出すようにしましょう（特に型のない言語の場合）

rubyでの実行例

$ ruby ex2。rb
Input weight (kg): 61。5
Input height (m) : 1。63
BMI: 23。147276

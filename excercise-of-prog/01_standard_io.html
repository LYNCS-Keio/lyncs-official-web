その1 標準入出力を扱う

文字を入力したり出力したりします。

必要な知識

  • 標準入出力の扱い方
  • 変数の宣言と代入のやりかた
    • 型のある言語（C言語、Javaなど）では基本的な型についての知識
      • 整数、文字、文字列など

問題1

何かしらの文字を出力してみましょう。

rubyでの実行例

$ ruby ex1.rb
hello world
$

問題2

入力した文字を受け取って、そのまま出力してみましょう。

rubyでの実行例

$ ruby ex2.rb
input something: hoge
hoge
$

問題3

以下のように年、月、日を入力し、それを表示するようにしてみましょう。

rubyでの実行例

$ ruby ex3.rb
Input year: 2013
Input month: 4
Input day: 16
Today is 2013/4/16. It is fine day today.
$

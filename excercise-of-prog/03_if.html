その3 条件分岐

特定の条件が満たされたときだけ処理を実行するようにします．

必要な知識

  • 条件分岐する方法
    • だいたい if
  • 条件を書く方法
    • == や != ，> などがとりあえず分かってればいい
  • orやandを表す方法
    • だいたい | と &
  • 複数の条件を書く方法
    • 言語によって else if や elsif などいろいろあってキレそうになる
  • 条件が満たされなかったときの処理を書く方法
    • だいたいelse

あったほうが幸せな知識

  • bool(boolean)型についての知識
    • bool型の値は if の条件に使えます
      • メソッドの返り値をそのまま条件に使える

問題1

整数を1つ読み込み，偶数か奇数かを表示するプログラムを作成しましょう．

rubyでの実行例

$ ruby ex1.rb
Input an integer: 1
1 is an odd number.
$ ruby ex1.rb
Input an integer: 2
2 is an even number.
$

問題2

整数を1つ読み込み，2および3で割り切れるかを表示するプログラムを作成しましょう．

ヒント

以下の4つの状態が考えられるので，それぞれについて別の出力を出せるように条件分岐を書いていきましょう．
順番に注意です．

  • 2だけで割り切れる
  • 3だけで割り切れる
  • 2でも3でも割り切れる
  • 2でも3でも割り切れない

rubyでの実行例

$ ruby ex2.rb
Input an integer: 14
14 can be divided by 2, but cannot be divided by 3.
$ ruby ex2.rb
Input an integer: 30
30 can be divided by both 2 and 3.
$ ruby ex2.rb
Input an integer: 19
19 cannot be divided by either 2 or 3.
$ ruby ex2.rb
Input an integer: 15
15 cannot be divided by 2, but can be divided by 3.
$

問題3

西暦で年を入力し，閏年か否かを判定するプログラムを作成しましょう．

ヒント

4で割れれば閏年，但し100で割れれば平年，但し400で割れれば閏年，ですね．
読みやすい書き方，短い書き方，いろいろな書き方があると思います．いろいろ考えてみるのもいいと思います．

Input year: 2013
2013 is a common year.

Input year: 2012
2012 is a leap year.

Input year: 2000
2000 is a common year.
